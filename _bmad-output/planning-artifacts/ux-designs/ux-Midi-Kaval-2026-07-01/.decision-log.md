# Decision Log

| Date | Decision | Rationale |
|------|----------|-----------|
| 2026-07-01 | EXPERIENCE.md is the primary deliverable; DESIGN.md is minimal (inherits Angular Material theme) | Kaval Online already has a mature Material theme. 2FA adds behavioral patterns, not a new visual identity. |
| 2026-07-01 | MatMenu triggered on click (not hover) for 2FA column | Reset 2FA is a sensitive action; accidental hover-opens would erode trust. Click is deliberate. |
| 2026-07-01 | Confirmation dialog default focus on Cancel button | Prevents accidental Enter-press confirmation of destructive 2FA reset. |
| 2026-07-01 | Bypass code copy uses silent tooltip, not MatSnackBar | Avoids drawing screen attention to a sensitive temporary code. Tooltip briefly confirms. |
| 2026-07-01 | TOTP input auto-submits on 6th character | Reduces friction for the most common enrollment step. Falls back to manual button for accessibility. |
| 2026-07-01 | Backup codes displayed as 2-column monospace grid on enrollment success | Scannable layout. Codes are for reference, not entry — so chips are appropriate. |
| 2026-07-01 | Backup code fallback appears after 1 failed TOTP attempt (not immediately) | Avoids cluttering the default TOTP screen. Single failure is the signal that the user likely doesn't have their app. |
| 2026-07-01 | Organisation toggles save immediately (no "Save" button) | Follows Material toggle pattern. Consistent with existing app behavior. |
| 2026-07-01 | Enrollment notification click navigates to Staff Management | Gives Director immediate context on which user enrolled. |
| 2026-07-01 | Key-screen mocks: Vendor Settings, Staff Management 2FA column, Confirmation Dialog, Bypass Code Dialog, Backup Codes display | These are the screens where layout drives behavior and anchors the visual language. All other screens reuse existing patterns. |
| 2026-07-01 | EXPERIENCE.md finalized — status set to final. Mockups promoted to mockups/. | All 5 key screens mocked; 3 remaining surfaces (login TOTP fallback, audit log, generic enrollment) spine-only. |
